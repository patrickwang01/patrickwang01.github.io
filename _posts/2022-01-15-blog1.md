To make the data frame much simpler and easier to read, we will create a function that will clean up the data frame:

If we run the code block below:
```python
def temps_df_prep(df):
    '''
    Cleans/simplifies temps.csv data frame for easier use

    Parameters:
    df = dataframe

    Output:
    Returns the updated temps data frame
    '''

    df = df.set_index(keys = ["ID", "Year"])
    df = df.stack()  # stack all the VALUES in 1 column
    df = df.reset_index()  # bring back ID and Year columns
    df = df.rename(columns = {"level_2"  : "Month" , 0 : "Temp"})
    df["Month"] = df["Month"].str[5:].astype(int)
    df["Temp"]  = df["Temp"] / 100
    
    return(df)
```
Running this function will remove all the `VALUES#` columns and stack them all into one column called `Temp`. The `VALUE` columns will also be stacked into a new column called `Month`.

## Writing a Query Function

For our example query function, we will design one that takes in a country name, a beginning and ending year range, and a specified month as parameters. Our query command will take in these specific arguments and filter through our 3 tables accordingly. The results of the query will be converted into a Pandas data frame for easy visualization and manipulation later on!


If we run the code block below:

```python
def query_climate_database(country, year_begin, year_end, month):
    '''
    Function that queries database consisting of 3 tables with user-specified parameters 
    and outputs a pandas dataframe that includes various measurements like locations, average temperature, etc. 
    
    Parameters:
    
    country = user-selected country
    year_begin = beginning of period range
    year_end = end of period range
    month = user-selected month
    
    Outputs:
    
    returns a pandas dataframe with relevant measurements
    '''
    
    # opens connection to the database with our 3 tables
    connection = sqlite3.connect("data.db")
    
    # set SQL query string that we will execute later 
    query = \
    '''
    SELECT S.name, S.latitude, S.longitude, C.name, T.year, T.month, T.temp
    FROM temperatures T
    LEFT JOIN stations S ON T.id = S.id
    LEFT JOIN countries C ON S.'fips 10-4' = C.'fips 10-4'
    WHERE (C.name = ?) AND (T.year BETWEEN ? AND ?) AND (T.month = ?)
    '''
    
    # transform results of query into pandas dataframe
    dataframe = pd.read_sql_query(query, conn, params = (country, year_begin, year_end, month)) 
    
    # close connection
    connection.close() 

    return dataframe
```

Our output will be a Pandas data frame based on our specified parameters. If we look closer into the function, we see the `connection.close()` command. A rule of thumb: we should always close the connection to our database after we finish using it!

## Creating a Geographic Scatter Function

Using our new query function, we can now make pretty and informative data visualization figures. We can also answer a lot of questions using with specific queries. For instance, our example query can answer the question: **How does the average yearly change in temperature vary within a given country?** 

Our query already provides the stations within a given country and we can calculate the yearly change in temperature pretty easily with the `Temps` column. 

<br> 

To begin, we should import the `plotly express` package within `plotly`
```python
from plotly import express as px
```

Now, we could first create a function that calculates the average change in the temperature every year using a basic linear regression. The coefficient of a linear regression is the slope or change in y per change in x, so if we regress temperature on year, the coefficient would represent the yearly change in temperature. 

If we run the code block below:

```python
from sklearn.linear_model import LinearRegression

# calculate yearly change in temp

def avg_change_coef(data_frame):
    '''
    Function that regresses temperature on year and returns the coefficient of the linear regression

    Paramters:
    a data frame

    Outputs:
    the coefficient of a linear regression rounded to 4 decimal points
    '''

    x = data_frame[["Year"]] 
    y = data_frame["Temp"]   
    LR = LinearRegression()
    LR.fit(x, y)
    return round(LR.coef_[0], 4)
```
Our return will be the coefficient of a linear regression per year rounded to the fourth decimal point. With this function we could begin constructing our new function that will create our geographic scatter plot figure!

<br>

Using our query function along with our newly created function that calculates the average change in temperature, if we run the code block below:

```python
def temperature_coefficient_plot(country, year_begin, year_end, month, min_obs, **kwargs):
    '''
    Function that creates a geographic scatter plot using the original query function with specific parameters 

    Parameters:
    
    Same as query_climate_database function
    min_obs = minimum number of years of observations 
    **kwargs = additional keyword arguments to modify plotly figure

    Output:
    Returns the generated scatter plot
    '''

    # use query function with specific paramters
    df = query_climate_database(country, year_begin, year_end, month)
    
    # count number of observations per station 
    df["Observations"] = df.groupby("NAME")["Temp"].transform(len)
    
    # calculate yearly change in temp
    avg_coefs = df.groupby(["NAME", "Month", "LATITUDE", "LONGITUDE", "Observations"]).apply(avg_change_coef)
    avg_coefs = avg_coefs.reset_index()
    avg_coefs = avg_coefs.rename(columns = {0 : "Estimated Yearly Change (C)"})
    
    # filter out stations that don't have min. number of years of data
    final_df = avg_coefs[avg_coefs["Observations"] >= min_obs]
    
    # create plot using updated df
    figure = px.scatter_mapbox(final_df,
                      lat = "LATITUDE",
                      lon = "LONGITUDE",
                      hover_name = "NAME", 
                      hover_data = ["Estimated Yearly Change (C)"], # insert calculated coefficients
                      color = "Estimated Yearly Change (C)", # set color based on coefs
                      color_continuous_midpoint = 0, 
                      title = "Estimated Average Yearly Temperature Change for Stations in India during January, 1980 - 2020",
                      **kwargs)
    
    return figure
```
Our return should be a geographic scatterplot filtered on the parameters that goes into the query function inside. If we look closer into the function, we see the code `df["Observations"] = df.groupby("NAME")["Temp"].transform(len)`. Because we may want to specify a minimum number of years of data that a station should have, this line creates a new column in our data frame that counts the number of observations a station has. Afterwards, we see that we apply our `avg_change_coef` function to create a new column that we call `Estimated Yearly Change (C)` that stores our calculated coefficients per year. With this updated data frame with our two new columns, we call the `px.scatter_mapbox` function and input our data frame into this to create our geographic scatterplot. Notice that we use a lot of modifications to change the appearance of our figure. Using `**kwargs` allows us to make more modifications when we call this whole function later on!

<br> 

To check that we have succesfully designed our function, we will use the same parameters as before and use a minimum observation of 10 as well as a few additional keyword arguments. We will also use a preset colormap to spruce up our figure. Running the code block below:

```python

        mapbox_style = "carto-positron",
        color_continuous_scale = color_map)

```
